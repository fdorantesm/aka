Complete Command Reference

All 7 commands at your fingertips to master alias management.

aka add

Add a new alias. Example: aka add gs "git status"

aka list [pattern]

List aliases with filter. Example: aka list '*dev*'

aka apply

Apply aliases to your shell config.

aka export

Export aliases to a file for backup.

aka import

Import aliases from an external file.

aka remove

Remove an existing alias by name.

aka version

Show the current version of aka.
